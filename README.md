## 🚀 Getting Started

### Prerequisites

* Node.js
* npm

### Installation

```bash
git clone https://github.com/ssalih755-github.io/react-portfolio.git
cd react-portfolio
npm install
```

### Run Locally

```bash
npm start
```

This will launch the app at [http://localhost:3000](http://localhost:3000)

## 🔧 Scripts
